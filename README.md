# Top-down shooter on DOTS (Unity)

> portfolio project

![img.png](https://media.githubusercontent.com/media/leni8ec/portfolio-3-dots-shooter/main/img.png?raw=true)

## Features
- Player movement and shooting
- Enemy AI with spawn system
- Bullet collision detection
- Health system for player and enemies
- Simple UI for score and health display
- Enemy shooting mechanics
- Collision handling for bullets and enemies
- ~~Scoring system~~
- Game over condition

## Usage packages
- DOTS
- UiToolkit
